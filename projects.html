Implementation of Distributed Key-Value Store for a P2P Network

  • Implemented a scalable key-value store using the Chord Protocol for decentralized networks. The Chord Protocol simplifies key lookup, node joining, and leaving processes, ensuring consistent behavior.
  • Developed and deployed a distributed key-value store application in Go language, leveraging gRPC for seamless inter-node communication.
  • Conducted performance evaluations, demonstrating O(log N) scalability in fetch time as the key count increased from 100k to 1.6M on a network comprising 100 nodes.

Raft Consensus Algorithm Implementation

  • Developed a replicated state machine using Raft, a distributed consensus protocol, to ensure log consistency across replicas in the system.
  • Implemented log replication and leader election in Elixir to handle node failures, rigorously testing the implementation through simulated failures.

Parallelized Graph Algorithm Optimization for Shared Memory Architectures

  • Implemented a framework for parallelizing graph algorithms on shared memory machines using OpenMP and C++.
  • Optimized shared memory access and modification using reduction techniques and compare-and-swap operations instead of OpenMP critical directives.
  • Achieved significant speedups, including 8x for Bellman-Ford, 3x for BFS, and 7x for Connected Components algorithms on graphs with 1M nodes and 15M edges.

Equity Market Modeling and Analysis

  • Utilized raw data from NYSE-listed stocks (June 20, 2007, to September 20, 2007) to prepare for modeling, adjusting for splits, and cleaning data using the Brownlees and Gallo (2006) procedure.
  • Optimized cleaning parameters through experimentation, refining data quality.
  • Performed statistical hypothesis testing to validate data integrity before modeling.
  • Built and implemented an impact model inspired by Almgren et al.’s Direct Estimation of Equity Market Impact, utilizing non-linear regression to model parameters.
  • Conducted extensive hypothesis testing to ensure the accuracy and validity of tuned parameters.

Semantic Image Segmentation of Satellite Images using U-Nets

  • Developed multiple U-Net models employing various encoders for semantic segmentation of land cover from satellite images sourced from the Landcover.ai dataset.
  • Implemented effective data augmentation strategies to bolster model accuracy and alleviate overfitting.
  • Evaluated model performance using a variety of metrics such as the Jaccard Index, Precision, Recall, and F1 Score. Selected the best model with a ResNet-18 encoder, which achieved a 67% Jaccard Index.

Implementation of Monte Carlo Sampling Algorithms

  • Implemented exact sampling schemes like change of variables, inversion, and rejection to sample from diverse distributions. Validated samples via scatter plots and 2D histograms.
  • Implemented importance sampling algorithms with and without resampling methods (Bernoulli, Multinomial, symmetric) to generate samples from parameterized Gaussian distributions, with and without knowing the normalization constant. Conducted variance comparisons to determine the most effective resampling methods.
  • Utilized sequential importance sampling to generate Self-Avoiding Walks in d dimensions.
  • Implemented Markov Chain Monte Carlo schemes using Gibbs sampler and Metropolis-Hastings sampling methods to generate samples from the 2D Ising Model.
  • Generated samples from the nearest neighbor XY model using various Langevin Dynamics-based sampling techniques. Compared integrated autocorrelation times for each method.
  • Implemented stochastic Newton and Affine Invariant Ensemble schemes to generate samples from the Rosenbrock density in 2D.

Incorporating Causal Inference into Recommendation Systems

  • Employed Causal Inference to elucidate cause-effect dynamics between users and items, laying the foundation for a robust recommendation system.
  • Developed an exposure model to unveil latent confounding factors, leveraging diverse factor models such as Probabilistic Principal Component Analysis (PPCA), Hierarchical Poisson Factorization (HPF), and Deep Exponential Family (DEF) for inferring substitute confounders.
  • Integrated computed confounders into the outcome model to enhance the sophistication of recommendation modeling.
  • Conducted predictive checks to validate the accuracy of derived latent factor models and delved into the qualitative dimensions of recommendation via latent factors captured by the models.
